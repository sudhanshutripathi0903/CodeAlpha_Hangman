# Hangman Game in Python

This is a simple Hangman game developed using Python.
The program randomly selects a word and the user has to guess letters within limited attempts.

## Features
- Random word generation
- Letter guessing system
- Limited number of attempts
- Displays correct and incorrect guesses

## Technologies Used
- Python

## How to Run
1. Save the file as hangman.py
2. Run using:
   python hangman.py

## Game Rules
- Guess one letter at a time
- Repeated letters are not allowed
- Wrong guesses reduce attempts
- Guess the full word to win
